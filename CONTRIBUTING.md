# Contributing to MLS4RN

Thank you for your interest in contributing to MLS4RN.

This project is in early development. Contributions, feedback, bug reports, and documentation improvements are welcome.

## How to contribute

1. Open an issue to discuss larger changes before starting work.
2. Fork the repository.
3. Create a feature branch.
4. Make your changes.
5. Open a pull request with a clear explanation of what changed.

## Development status

The API and project structure may change while the prototype is being developed.

## Code of conduct

Please follow the project's Code of Conduct when participating.
